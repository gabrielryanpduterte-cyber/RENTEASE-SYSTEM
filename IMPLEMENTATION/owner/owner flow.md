# Owner Flow

## Landlord Workflow: End to End

### Phase 1: Setup First Login

Register as owner

→

Complete boarding house profile

→

Upload cover photo

→

Set house rules

Add rooms one by one

→

Upload room photos

→

Set amenities, rate, capacity

→

Rooms go live Available

### Phase 2: Reservation Management

Seeker submits reservation

→

Landlord gets pending alert

→

Reviews applicant + ID

Approve → room → Occupied + billing cycle created

or

Reject → add remarks → seeker notified

### Phase 3: Monthly Rent Cycle

Go to Rent Tracking

→

Click "Generate Bills" for month

→

Preview → Confirm

Billing cycles created for all active tenants

→

Tenants see "Unpaid" on their dashboard

Tenant pays + uploads proof

→

Landlord sees proof badge

→

Mark as Paid select method

→

Status → Paid ✓

### Phase 4: Reporting

Open Reports page

→

Income chart 12 months

+

Occupancy rate

+

Unpaid list

Select month range

→

Print / Export

### Phase 5: Room Lifecycle

Add room → Available

→

Reservation approved → Occupied

→

Tenant leaves → set Available again

Decommission room → Archive not delete

→

Hidden from public, history preserved
